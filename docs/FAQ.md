# FAQ

## Isso é open source do projeto?

Não. Este repositório é apenas documental e focado em arquitetura/operação para portfólio.

## Posso usar este material como base de estudo?

Sim, desde que sem reutilizar conteúdo proprietário ou sensível de terceiros.

## Por que não publicar o código?

Para preservar propriedade intelectual, segurança operacional e acordos de confidencialidade.

## O que um avaliador técnico deve olhar primeiro?

1. `ARCHITECTURE.md`
2. `DECISIONS.md`
3. `DIAGRAMS.md`
4. `OPERATION.md`
